Frequently Asked Questions

An accordion is a family of box-shaped musical instruments of the bellows-driven free-reed aerophone type ( producing sound as air flows past a reed in a frame). This FAQ's system of organization is also called an accordion in web-development because the way a section pops out after a question is clicked is similar to how an accordion is played.

CSS stands for Cascading Style Sheets and is a rule based language that is used to describe the presentation of documents written in other languages, like HTML.

A CSS framework is a lot like a library where you only need to read the work of other people to learn what they worked hard to learn. It essentially contains hundreds or even thousands of classes that allow for much easier and consistent designs.

RGB stands for Red, Green, and Blue; it is a way of specifying colour on a computer. Red, Green, and Blue are specifically chosen for this as they are primary colours and it has been proven that mixing certain amounts of each can produce almost all other colours.
